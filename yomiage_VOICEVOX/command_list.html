yomiage_VOICEVOXコマンド一覧

2022/03/13時点で実装されているコマンド一覧です。

基本コマンド

!help
コマンドリストが表示されます。（簡略版）
!hello
バージョン情報を確認できます。
!join
BOTをボイスチャンネルに呼びます。（コマンド使用者が入っているボイスチャンネルに飛びます）
!leave
BOTをボイスチャンネルから退場させます。
!stop
再生している音声をストップします。

単語リストコマンド

!wlist add A B
読み仮名の登録をします。AをBと読ませます。
例）!wlist add gorilla ゴリラ
!wlist delete A
辞書登録の削除を行います。
例）!wlist delete gorilla
!wlist show
辞書登録の確認を行います。

調声コマンド

!chg_my_voice
ボイスを変えます。!chg_my_voice A B [C]で設定できます。
Aは話者名、Bはスタイル名です。使用できる話者とスタイルは!show_speakersコマンドで確認してください。
Cはソフトウェア名です。VOICEVOX、COEIROINK、LMROID、SHAREVOXから選んで入力してください。
Cはなくてもコマンドを受け付けます。将来的に、複数のソフトで同じ話者名とスタイル名の組み合わせが出た場合に向けての対応となります。
!chg_voice_setting
スタイル毎に話速・音高・抑揚・音量を変更します。!chg_voice_setting A B C D で設定できます。
Aは話者名、Bはスタイル名です。使用できる話者とスタイルは!show_speakersコマンドで確認してください。
Cは speed pitch intonation volume から選んで入力してください。それぞれ話速・音高・抑揚・音量に対応します。
Dは設定値です。speedは0.5～2.0、pitchは-0.15～0.15、intonationとvolumeは0～2.0の範囲で入力してください。

各種設定

!inform_someone_come
入退室通知のオンオフを切り替える
!inform_tmp_room/
botが入室しているボイスチャンネル名のオンオフを切り替える
!time_signal
時報のオンオフを切り替える
!read_name
名前読み上げ機能のオンオフを切り替える
!number_of_people
在室人数チェックのオンオフを切り替える
!auto_leave
自動退出のオンオフを切り替える
!word_count_limit
文字数制限の設定
例: 50文字に設定したい時）!word_count_limit 50
!show_setting
現在の設定を確認する
!show_speakers
現在利用できるソフトウェア、話者、スタイルの一覧を表示する
